# AIRI

## 環境準備
1. Python 3.7 インストール
2. AIRIディレクトリに移動
3. `pip install pipenv`
4. `pipenv install`
5. api/DDLディレクトリに移動
6. 01_create_table.sh を実行
7. 02_insert_user_sampledata.sh を実行


## 実行手順
1. apiディレクトリに移動
2. `pip run python server.py`
3. appディレクトリに移動
4. `pip run python server.py`
5. ブラウザでURLを開く http://localhost:8080
